AI Support Agent using n8n

## Project Overview

This project was developed as part of the Codebasics AI Product Management Cohort. The objective was to build an AI-powered customer support workflow using n8n that automatically answers customer queries using a knowledge base.

## Problem Statement

Customer support teams receive repetitive questions related to course access, refunds, certificates, payments, and account issues. Manually responding to these queries is time-consuming and does not scale well.

## Solution

The n8n workflow automates the customer support process by:

1. Receiving a customer query.
2. Searching the support knowledge base.
3. Sending the relevant context to an AI model.
4. Generating an accurate response.
5. Returning the response to the customer.

## Workflow

Customer Query
↓
Knowledge Base Search
↓
AI Model
↓
Generate Response
↓
Customer Reply
